# ⚽ GoalHub

![GoalHub](Goalhub.png)

## Descripción

GoalHub es una aplicación web Full Stack centrada en el fútbol europeo.

### Funcionalidades

- 🏆 Ligas europeas
- 📅 Calendario de partidos
- 🎯 Quiniela
- 🥇 Ranking de usuarios
- 📰 Noticias
- 🛒 Tienda online

## Tecnologías

- React
- JavaScript
- Python
- Flask
- SQL
- SQLAlchemy
- REST API
