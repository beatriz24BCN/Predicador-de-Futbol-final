# ⚽ GoalHub

![GoalHub](Goalhub.png)



## Description

GoalHub is a Full Stack web application focused on European football. It allows users to follow competitions, predict match results, stay up to date with football news, and compete with other users through rankings.

---

## Features

🏆 European Football Leagues

📅 Match Schedule & Fixtures

🎯 Match Predictions (Quiniela)

🥇 User Rankings

📰 Football News

🛒 Online Store

🔐 User Authentication

📱 Responsive Design

## Project Goals

- Build a modern Full Stack football platform.
- Consume external football APIs.
- Implement user authentication and database management.
- Create an intuitive and responsive user experience.
- Develop a scalable architecture using React and Flask.
